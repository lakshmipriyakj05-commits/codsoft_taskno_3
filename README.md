# Calculator Application

## Overview

This project is a basic calculator built using HTML, CSS, and JavaScript. It performs arithmetic operations such as addition, subtraction, multiplication, and division through an interactive user interface.

## Features

* User-friendly calculator interface
* Addition, subtraction, multiplication, and division
* Clear button functionality
* Real-time display updates
* Responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

calculator/
│
├── index.html
├── style.css
└── script.js

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Use the calculator to perform calculations.

## Learning Outcomes

* JavaScript event handling
* DOM manipulation
* CSS Grid layout
* Arithmetic operations using JavaScript
